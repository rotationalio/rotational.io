Open Source Projects & Publications

Open source software, continuous learning, and research are vital to software development and innovation . Learn about our open source projects and publications.

Our Open Source Projects

  • Yellowbrick

    Yellowbrick is an open source project built using Python that extends the scikit-learn API with visual analysis and diagnostic tools. The Yellowbrick library is a diagnostic visualization platform for machine learning.

  • TRISA

    The Travel Rule Information Sharing Alliance (TRISA) is an open source standard that defines how Virtual Asset Service Providers (VASPs) securely exchange digital assets to comply with the FATF Travel Rule while protecting privacy.

  • Whisper

    Whisper is a secure secret-sharing utility that allows users to securely share and control access to confidential information such as passwords, documents, config files, keys, and certificates across an organization.
